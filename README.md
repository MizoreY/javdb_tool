# JavDB 番号评分刮削工具

自动从 JavDB 查询番号评分，更新 Emby/Jellyfin 的 NFO 文件。

## 功能

- 全量扫描：扫描文件夹内所有番号并更新评分
- 补全模式：补全今日未更新或评分缺失的番号
- Cloudflare 重试：触发验证时自动等待重试，连续 3 次才判定封禁
- 断点续传：进度自动保存，中断后可继续
- 导出结果：支持 CSV / JSON 格式导出

## 评分规则

- `rating` = 评分四舍五入保留一位小数
- `criticrating` = 评分 × 20 保留一位小数

## 使用方法

### CLI 版

1. 双击 `启动.bat`
2. 选择模式（1 或 2）
3. 输入视频文件夹路径
4. 确认开始（y）
5. 等待查询完成
6. 选择是否导出

### GUI 版

1. 双击 `启动GUI.bat`
2. 点击「浏览」选择文件夹
3. 选择模式
4. 点击「开始刮削」

## 环境变量（可选）

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `JAVDB_PROXY` | 代理地址 | `http://127.0.0.1:7890` |
| `JAVDB_DELAY_MIN` | 最小间隔秒数 | `8` |
| `JAVDB_DELAY_MAX` | 最大间隔秒数 | `15` |

## 依赖

- Python 3.9+
- Chrome 浏览器
- 代理（访问 JavDB）

```bash
pip install nodriver beautifulsoup4
```

## 注意事项

- 修改 NFO 后需在 Emby/Jellyfin 中刷新媒体库
- 连续 3 次触发 Cloudflare 验证会暂停，需手动在浏览器验证后继续
